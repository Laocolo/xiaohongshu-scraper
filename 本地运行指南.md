# 小红书爬虫 - 本地运行指南

## 📦 解压后目录结构

```
xiaohongshu_scraper/
├── xhs_scraper.py              # 基础版（模拟数据演示）
├── xhs_scraper_real.py         # 真实爬取版 ⭐推荐
├── xhs_scraper_search.py       # 搜索引擎版
├── xhs_scraper_selenium.py     # Playwright浏览器版
├── proxies.txt.example         # 代理配置模板
├── xhs_cookies.txt.example     # Cookie配置模板
├── README.md                   # 项目文档
└── output/                     # 输出目录
```

## 🚀 快速开始（3步）

### 第1步：安装 Python 依赖

确保已安装 Python 3.8+，然后打开命令行运行：

```bash
pip install requests beautifulsoup4 pandas jieba snownlp lxml openpyxl
```

### 第2步：配置 Cookie（重要！）

1. 用浏览器打开 https://www.xiaohongshu.com 并登录
2. 按 F12 打开开发者工具
3. 切换到 Network（网络）标签
4. 刷新页面（F5）
5. 点击任意请求 → Headers → 找到 `cookie:` 
6. 复制整个 Cookie 值
7. 创建文件 `xhs_cookies.txt`，粘贴 Cookie 保存

### 第3步：运行爬虫

```bash
python xhs_scraper_real.py
```

## 📊 输出文件

运行完成后在 `output/` 目录查看：
- `xhs_real_data_xxx.json` - JSON数据
- `xhs_real_data_xxx.xlsx` - Excel数据  
- `report_real_xxx.md` - 分析报告

## ⚙️ 修改爬取目标

编辑 `xhs_scraper_real.py` 文件顶部的 `Config` 类：

```python
class Config:
    # 修改这里的关键词
    KEYWORDS = [
        "飞鹤奶粉",
        "爱他美",
        # 添加你需要的品牌...
    ]
    
    # 修改每个关键词爬取数量
    NOTES_PER_KEYWORD = 50
```

## 🛡️ 如果还是被封

### 方案1：使用代理
1. 复制 `proxies.txt.example` 为 `proxies.txt`
2. 每行填一个代理IP，格式：`192.168.1.1:8080`
3. 修改代码中 `USE_PROXY_FILE = True`

### 方案2：使用 Playwright 版（最稳定）
```bash
# 安装 Playwright
pip install playwright
playwright install chromium

# 运行
python xhs_scraper_selenium.py
```

## ❓ 常见问题

**Q: 提示 404 或请求失败？**
A: Cookie 过期或 IP 被封，重新获取 Cookie 或等待一段时间

**Q: 想要更多数据？**
A: 修改 `NOTES_PER_KEYWORD` 参数，或多次运行

**Q: 数据是真实的吗？**
A: 报告中会显示 `真实数据: X条` 和 `模拟数据: X条`
